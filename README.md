# Claw Legal Counsel Skills

This repository contains OpenClaw skills for legal consultation and OCR workflows.

## OpenClaw Natural-Language Install Guide

Use this pure natural-language instruction in an OpenClaw chat:

```text
帮我从 GitHub 仓库 wangruitiancc/Claw-Legal-Counsel 安装 law/skills/legal-counsel 和 law/skills/paddle-ocr-pdf-md 到 OpenClaw 的 skills 目录，安装完成后提醒我重启 OpenClaw。
```

Optional explicit version (more deterministic):

```text
请使用 $skill-installer 帮我从 GitHub 安装两个技能：
仓库：wangruitiancc/Claw-Legal-Counsel
路径：
1) law/skills/legal-counsel
2) law/skills/paddle-ocr-pdf-md
安装到默认 skills 目录，然后告诉我安装结果，并提醒我重启 OpenClaw。
```
